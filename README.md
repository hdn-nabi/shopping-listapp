# 🛒 쇼핑 리스트 앱

항목 추가·완료·삭제 기능을 갖춘 심플한 한국어 쇼핑 리스트 웹 앱입니다.

## 특징

- 단일 HTML 파일 (`index.html`) — 별도 설치·빌드 불필요
- `localStorage` 기반 데이터 저장 (브라우저 새로고침 후에도 유지)
- 항목 추가 / 완료 체크 / 개별 삭제 / 완료 항목 일괄 삭제
- 슬라이드 인/아웃 애니메이션
- 반응형 디자인 (모바일 친화적)

## 사용 방법

`index.html` 파일을 브라우저에서 바로 열어 사용합니다.

```bash
git clone https://github.com/hdn-nabi/shopping-listapp.git
open shopping-listapp/index.html
```

## 기술 스택

- HTML5 / CSS3 / Vanilla JavaScript
- Web Storage API (localStorage)
